# Simulador de Entregas com Drones - Desafio dti

Este projeto é uma API RESTful desenvolvida em **Node.js** para gerenciar e otimizar a logística de entregas por drones.

## Tecnologias Utilizadas
- Node.js
- Express (API)
- Jest (Testes Unitários)
